## 头部参数

example：

```markdown
title: test-codestyle
comments: true
tags:
  - hexo
date: 2019-10-31 11:39:35
toc: ture
declare: true
```



1. 是否开启字数统计
   - 默认开启
   - 关闭：`no_word_count: true`
2. 是否添加版权声明
   - 默认关闭
   - 开启：`declare: true`
3. 是否启用目录
   - 默认关闭
   - 开启：`toc: true`

4. 是否开启评论
   - 默认不开启
   - 开启：`comments: true`
